How to Solve...

Algebra is just like a puzzle where we start with something like "x − 2 = 4" and we want to end up with something like "x = 6". But instead of saying "obviously x=6", use this neat step-by-step approach:

1. Work out what to remove to get "x = ..."
2. Remove it by doing the opposite (adding is the opposite of subtracting)
3. Do that to both sides


Example:

We want to remove the "-2" in: x -2 =4


x - 2 + 2 = 4 + 2

Which makes

x + 0 = 6

x = 6
